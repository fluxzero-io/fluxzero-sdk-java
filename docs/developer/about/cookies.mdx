**Last update:** July 2026

At Fluxzero, your privacy matters. This page explains how the public Fluxzero website and documentation use cookies and similar browser technologies.

## Summary

The public Fluxzero website and documentation only use cookies when they are necessary for the website to work or for functionality you explicitly use, such as signing in with GitHub to submit documentation feedback.

We do not use optional cookies on the public website. That means we do not currently show a cookie banner.

## What Are Cookies?

Cookies are small text files stored on your device when you visit a website. They can be used to keep a session secure, remember a technical choice, or support a feature you requested.

Similar browser technologies, such as local storage and session storage, can store small pieces of information in your browser without using cookies.

## Cookies We Use

### GitHub Feedback Login Cookies

When you sign in with GitHub to submit documentation feedback, Fluxzero uses strictly necessary cookies to complete that process securely.

| Cookie | Purpose | Expiry |
| --- | --- | --- |
| `fx_gh_state` | Confirms that the GitHub sign-in was started from your browser. | 10 minutes, or deleted immediately after a successful sign-in |
| `fx_gh_auth` | Keeps you signed in while you submit documentation feedback. | Until the GitHub sign-in expires, or 24 hours if no expiry is provided |

These cookies are kept to the minimum necessary and are never used to track your activity.

## Similar Technologies

The documentation may use browser storage for functional user-interface preferences:

| Storage item | Purpose |
| --- | --- |
| `starlight-theme` | Remembers your light/dark/auto theme preference in the documentation. |
| `sl-sidebar-state` | Remembers the expanded/collapsed state of the documentation sidebar. |
| `didyouknow-dismissed` | Remembers whether you dismissed an informational documentation tip. |
| `fz_feedback_quote`, `fz_feedback_draft`, `fz_feedback_rect` | Temporarily preserves a documentation feedback draft during GitHub sign-in. |

These storage items are used to provide the website or documentation feature you are using. They are not used for advertising or cross-site tracking.

## Non-Essential Cookies

We do not use cookies or similar browser technologies that are not necessary for the website or for a feature you choose to use. In particular, we do not use them for advertising, retargeting, cross-site profiling, social media tracking, heatmaps, session replay or optional analytics on the public website.

If we decide to introduce optional analytics or similar non-essential technologies in the future, we will update this notice and, where required, ask for consent before using them.

## Managing Your Choices

You can block or delete cookies and browser storage through your browser settings. Some features, such as GitHub sign-in for documentation feedback, may not work correctly if necessary cookies are blocked.

## Questions?

If you have any questions about this policy or how we use cookies, contact us at [privacy@fluxzero.io](mailto:privacy@fluxzero.io).
